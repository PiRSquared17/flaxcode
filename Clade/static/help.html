Help

Taxonomies

To select a taxonomy, use the drop-down menu at top left. The taxonomy will appear, unexpanded, in the left-hand part of the screen. You can click on the triangular icon at the left of a taxonomy element to expand or contract a node. Clicking on the document icon next to the node name will bring up a context menu (none of the options are currently functional in the prototype).

Click on a taxonomy node to show documents classified under the Node represented by a node. Documents are classified using the list of Keywords contained in a Node.

Keywords & Suggested Keywords

Keywords for a selected Node are shown in the area titled 'Keywords'. These keywords are used by the categorisation system to file a document under a particular taxonomy node (or nodes). Extra keywords, automatically suggested by the system based on document content, are shown to the right of the Keywords area in the area titled 'Suggestions' - these are not used by the categorisation system.

Keywords have weights - in the prototype these default to 100%.

Adding a new keyword

To add a new keyword click the + icon. An editable field will appear highlighted in yellow. Enter the new keyword data and click outside the editable field to finish.

Editing a keyword

Click the keyword to edit its text.

Removing a keyword

Keywords may be removed by clicking the small 'x' icon next to the keyword text.

Adding a suggested keyword

To add a suggested keyword for use by the categorisation system, click on it in the Suggestions area and drag it across to the Keywords area. Document categorisation will repeat based on the new list of keywords.

Documents

The 'Documents' area shows which documents are categorised under the currently selected Node or taxonomy node. This list will change when the Keywords for a particular Node are modified. When Keywords change, the order of documents also changes: the New and Old order is shown numerically.

Displaying a document

To show the detail of document click its title or ID number. You will see the Keywords this document contains; the Metadata (author, title etc.) for this document and the document text, with relevant keywords highlighted (note that full formatting is not preserved). You will also see at the left the Nodes which the document has been classified against. Click the node name highlighted in blue at top right to return to the taxonomy.
