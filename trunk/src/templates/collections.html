Create a new collection

Current document collections are listed above. The columns are as follows:

  1. A link to delete the collection in that row. This link takes you to a page confirming that you wish to actually delete the collection.
  2. The name of the collection and a link to the detail page for that collection.
  3. The description of the collection.
Note finally the last row. This allows you to enter data for a new collection and create it.
