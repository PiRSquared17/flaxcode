Flax Site Search

Version .

Flax Site Search provides a basic search facility for websites and is based on Lemur Consulting's search solution platform, Flax. Further support is available at the Flax website.

Flax is open source software, with no license fees, meaning that its source code is freely available. Flax is based on Xapian, a high-performance, cross-platform and extremely accurate technology used across the world.

The Flax search solution platform allows for the development of advanced search systems with features that include:

  • Boolean search operators (AND, OR, NOT), probabilistic Bayesian, wildcard, phrase and proximity search
  • Allow users to refine their search by auto-suggesting extra search terms & related documents
  • Social network and 'Web 2.0' features including faceted search and tags
  • Spelling correction (automatic or dictionary-driven) and thesaurus
  • Indexes of hundreds of millions of documents
  • Customisable output formats using HTML templates and/or XML
  • Custom metadata and fields to aid searching
  • Rank results by relevance, date or by field
  • Intelligent, adaptive crawling of websites
  • Integration with third-party databases, email systems and file repositories
  • Integration with security architectures
  • Automatic backup, index validation and monitoring
